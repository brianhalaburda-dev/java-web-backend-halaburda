### Primera clase
- Instalamos JDK y verificamos con java -version
- Instamos Git y verificamos con git --version
- Descargamos e instalamos el IDE, lo tenemos funcionando
- Creamos el repositorio en GitHub
- Creamos un proyecto Spring Boot (con las librerías Spring Web y Rest Repositories)
- Configuramos el archivo application.properties para cambiar de puerto (8686) y el context (/backend)
- Creamos un controller con un endpoint GET básico
- Ejecutamos la aplicación localmente
- Probamos el endpoint utilizando un REST Client (Insomnia), lo tenemos funcionando
- Realizamos un commit y push al repositorio (algunos crearon sus tokens)
- Actualizamos el README file
